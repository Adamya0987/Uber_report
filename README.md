# 🚖 Uber Data Analysis Dashboard | Power BI

An interactive **Power BI dashboard** built to analyze Uber ride bookings, revenue, customer behavior, vehicle performance, cancellations, and pickup locations. The dashboard provides business insights through dynamic visualizations, KPIs, and drill-down analysis to help stakeholders monitor operational performance and make data-driven decisions.

---

## 📌 Project Overview

This project transforms raw Uber ride data into an interactive business intelligence dashboard using **Power BI**. It helps analyze ride trends, booking performance, revenue, customer preferences, and operational efficiency through multiple report pages.

## 📊 Business Insights

- Auto generated the highest revenue among all vehicle categories.
- UPI was the most preferred payment method.
- Monthly booking trends highlighted seasonal demand variations.
- Customer cancellation reasons helped identify operational bottlenecks.
- Pickup and drop location analysis identified high-demand areas.
- Vehicle performance comparison helped evaluate fleet efficiency.
